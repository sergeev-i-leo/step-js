Простая вёрстка

Для начинающих jsx, компоненты ui можно собирать "пошагово" - так, как это делается в JavaFX, Swing, Ink и т.п.

Такой подход упрощает условную вёрстку.

Архитектура Flux

Приложение поддерживает много глобальных состояний. Пользовательский интерфейс можно 'нарезать' на самостоятельные 'куски', разрабатывать и тестировать их независимо друг от друга.

Использование компонентов Bootstrap

Компоненты Bootstrap работают "из коробки". Это позволяет быстро получать работающее приложение, которое потом можно стилизовать по брендбуку.
